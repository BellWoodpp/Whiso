# Privacy Policy / 隐私权政策 / プライバシーポリシー

Effective date / 生效日期：2025-12-27

## English

**WHOIS Domain Date** shows WHOIS creation dates for domains on the page you are viewing.

### When this extension runs

- The extension only scans the current page after you click the extension’s toolbar icon (to show the panel).

### Data processed

- **Domain names** detected from the current page.
  - `SITE` mode: only the current website’s registrable domain (e.g. `example.com`).
  - `ALL` mode: additional domains found in page links/resources/text.

### Network requests / data sharing

- To fetch WHOIS creation dates, the extension sends **domain names** to `https://whois.freeaiapi.xyz/`.
- If the WHOIS service does not support a TLD, the extension may also send **domain names** to `https://rdap.org/` and/or the authoritative RDAP server for that TLD (as part of standard RDAP redirects).
- To validate TLDs (reduce false positives), the extension may download the public IANA TLD list from `https://data.iana.org/` and cache it locally.
- The extension does **not** send page titles, full URLs, form inputs, cookies, passwords, or any personal identifiers.
- Like any standard web request, the WHOIS service may receive technical data such as your IP address and User-Agent as part of the HTTP connection. The extension does not add any extra identifiers to those requests.

### Local storage

- The extension stores UI settings (language, scope, panel position) and a cache of WHOIS results **locally** using `chrome.storage.local`.
- Cache retention: up to **30 days** (then it is refreshed as needed).

### Your choices

- You can remove locally stored data by clearing the extension’s site data / extension storage, or by uninstalling the extension.

### Selling / advertising

- We do not sell user data.
- We do not use the data for advertising or tracking.

- If we update this policy, we will update the effective date above.

## 中文

**WHOIS 域名时间**用于显示你正在查看的网页中域名的 WHOIS 注册/创建日期。

### 扩展何时运行

- 扩展仅在你点击扩展工具栏图标（显示面板）后，才会扫描当前页面。

### 处理的数据

- 从当前页面中识别到的**域名**。
  - `SITE` 模式：仅当前站点的可注册主域名（例如 `example.com`）。
  - `ALL` 模式：还会包含页面链接/资源/文本里的其他域名。

### 网络请求 / 数据共享

- 为查询 WHOIS 创建日期，扩展会将**域名**发送到 `https://whois.freeaiapi.xyz/`。
- 若该 WHOIS 服务不支持某些后缀，扩展也可能会将**域名**发送到 `https://rdap.org/` 以及该后缀对应的权威 RDAP 服务（RDAP 标准重定向的一部分）。
- 为校验域名后缀（减少误识别），扩展可能会从 `https://data.iana.org/` 下载公开的 IANA TLD 列表并缓存在本地。
- 扩展不会发送网页标题、完整 URL、表单输入、Cookie、密码或任何可用于识别个人身份的信息。
- 与任何标准网络请求类似，WHOIS 服务端在建立连接时可能会接收到 IP 地址、User-Agent 等技术信息。扩展不会在请求中额外添加任何用于识别用户的标识。

### 本地存储

- 扩展会使用 `chrome.storage.local` 在本地保存设置（语言、范围、面板位置）以及 WHOIS 查询缓存。
- 缓存保留：最长 **30 天**（到期后按需刷新）。

### 你的选择

- 你可以通过清除扩展存储数据，或卸载扩展来删除本地保存的数据。

### 出售 / 广告

- 我们不会出售用户数据。
- 我们不会将数据用于广告或跟踪。

## 日本語

**WHOIS Domain Date** は、閲覧中のページに含まれるドメインの WHOIS 作成日を表示します。

### 拡張機能が動作するタイミング

- ツールバーの拡張機能アイコンをクリックしてパネルを表示した後にのみ、現在のページをスキャンします。

### 処理するデータ

- 現在のページから検出した **ドメイン名**
  - `SITE` モード：現在のサイトの登録可能なドメイン（例：`example.com`）のみ
  - `ALL` モード：ページのリンク/リソース/本文テキストから見つかった追加ドメインも対象

### ネットワークリクエスト / データ共有

- WHOIS 作成日取得のため、拡張機能は **ドメイン名** を `https://whois.freeaiapi.xyz/` に送信します。
- WHOIS サービスが未対応の TLD の場合、拡張機能は **ドメイン名** を `https://rdap.org/` および/または当該 TLD の権威 RDAP サーバーに送信することがあります（RDAP の標準リダイレクトの一部）。
- TLD を検証し誤検出を減らすため、拡張機能は `https://data.iana.org/` から公開 IANA TLD リストをダウンロードし、ローカルにキャッシュする場合があります。
- 拡張機能はページタイトル、完全な URL、フォーム入力、Cookie、パスワード、その他個人を特定し得る情報を送信しません。
- 標準的な Web リクエストと同様に、WHOIS サービスは HTTP 接続の一部として IP アドレスや User-Agent 等の技術情報を受け取る場合があります。拡張機能はそれらのリクエストに追加の識別子を付与しません。

### ローカル保存

- 拡張機能は UI 設定（言語、範囲、パネル位置）と WHOIS 結果キャッシュを `chrome.storage.local` に **ローカル** 保存します。
- キャッシュ保持期間：最大 **30 日**（必要に応じて更新）

### あなたの選択

- 拡張機能のサイトデータ / 拡張機能ストレージを消去するか、アンインストールすることでローカル保存データを削除できます。

### 販売 / 広告

- ユーザーデータを販売しません。
- 広告やトラッキング目的で使用しません。
