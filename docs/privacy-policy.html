Privacy Policy / 隐私权政策

Effective date / 生效日期：2025-12-25

English

WHOIS Domain Date shows WHOIS creation dates for domains on the page you are viewing.

When this extension runs

  • The extension only scans the current page after you click the extension’s toolbar icon (to show the panel).

Data processed

  • Domain names detected from the current page.
    • SITE mode: only the current website’s registrable domain (e.g. example.com).
    • ALL mode: additional domains found in page links/resources/text.

Network requests / data sharing

  • To fetch WHOIS creation dates, the extension sends domain names to https://whois.freeaiapi.xyz/.
  • The extension does not send page titles, full URLs, form inputs, cookies, passwords, or any personal identifiers.
  • Like any standard web request, the WHOIS service may receive technical data such as your IP address and User-Agent as part of the HTTP connection. The extension does not add any extra identifiers to those requests.

Local storage

  • The extension stores UI settings (language, scope, panel position) and a cache of WHOIS results locally using chrome.storage.local.
  • Cache retention: up to 30 days (then it is refreshed as needed).

Your choices

  • You can remove locally stored data by clearing the extension’s site data / extension storage, or by uninstalling the extension.

Selling / advertising

  • We do not sell user data.
  • We do not use the data for advertising or tracking.

Contact

  • Contact: use the developer contact email shown on the Chrome Web Store listing.

Changes

  • If we update this policy, we will update the effective date above.

中文

WHOIS 域名时间用于显示你正在查看的网页中域名的 WHOIS 注册/创建日期。

扩展何时运行

  • 扩展仅在你点击扩展工具栏图标（显示面板）后，才会扫描当前页面。

处理的数据

  • 从当前页面中识别到的域名。
    • SITE 模式：仅当前站点的可注册主域名（例如 example.com）。
    • ALL 模式：还会包含页面链接/资源/文本里的其他域名。

网络请求 / 数据共享

  • 为查询 WHOIS 创建日期，扩展会将域名发送到 https://whois.freeaiapi.xyz/。
  • 扩展不会发送网页标题、完整 URL、表单输入、Cookie、密码或任何可用于识别个人身份的信息。
  • 与任何标准网络请求类似，WHOIS 服务端在建立连接时可能会接收到 IP 地址、User-Agent 等技术信息。扩展不会在请求中额外添加任何用于识别用户的标识。

本地存储

  • 扩展会使用 chrome.storage.local 在本地保存设置（语言、范围、面板位置）以及 WHOIS 查询缓存。
  • 缓存保留：最长 30 天（到期后按需刷新）。

你的选择

  • 你可以通过清除扩展存储数据，或卸载扩展来删除本地保存的数据。

出售 / 广告

  • 我们不会出售用户数据。
  • 我们不会将数据用于广告或跟踪。

联系方式

  • 联系方式：请使用 Chrome Web Store 扩展详情页中显示的开发者联系邮箱。

更新

  • 若本政策更新，我们将更新上方的生效日期。
